**请求**
```json
{
	"type":11121,
	"data":{
		"user_id":"78812998xxxxxx",
		"corp_id":"0",
		"from_user_id":"7881299524xxxx",
		"verify":"你好"
	}
}
```

> user_id 在接收名片消息中取，user_id
> corp_id 在接收名片消息中取, corp_id
> from_user_id 是接收名片消息中的sender
> verify 验证消息

**返回**

```json
{
	"data":{
		"error_code":0,
		"error_message":"ok"
	},
	"type":11121
}
```
